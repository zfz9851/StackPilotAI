# StackPilot AI（AI 研发助手）使用说明

## 1. 启动

```bash
npm install
npm run dev
```

- 前端地址：`http://127.0.0.1:5173`
- 后端地址：`http://127.0.0.1:3000`

## 2. 配置流程

1. 在左侧菜单进入“项目基础”，填写项目名称、描述、类型、开发语言。
2. 依次在“前端选型 / 后端选型 / 数据层 / 运维部署 / 架构设计”勾选方案。
3. 底部预览区实时查看 JSON（结构化配置）、Markdown（文档）、文本、Prompt。

## 3. 导出与复制

- 顶部“导出”支持：JSON、Markdown、TXT。
- “复制 Prompt”可直接复制 AI 提示词。

## 4. 选型库管理

进入“选型库管理”后可：

- 新增自定义选项
- 编辑选项名称/描述
- 启用/禁用选项
- 删除选项（内置项删除会自动转为禁用）
- 恢复默认选型库
- 导出选型库 JSON（团队共享）
- 导入选型库 JSON

## 5. 持久化

- 所有配置与选型库存入本地 `SQLite（轻量数据库）` 文件：`server/stackpilot.db`
- 页面刷新后自动恢复最新配置

## 6. 异常处理

- 配置损坏：后端自动回退默认配置
- 非法字符：自动清洗控制字符
- 超长输入：项目名称/描述/类型/语言均有限长
- 导出失败：前端显示失败提示，可重试
