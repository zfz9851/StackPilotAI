# StackPilot AI

- 项目目标：基于图形化配置生成全栈技术方案、结构化 JSON、Markdown、TXT 与 AI Prompt。
- 前端栈：`Vue2.x（第二代 Vue 框架）` + `Element-UI（界面组件库）`
- 数据持久化：`SQLite（轻量数据库）` 本地文件

## 快速开始

```bash
npm install
npm run dev
```
